📊 Дизайнер отчётов Fast Reports

В состав My-Business CRM входит мощный инструмент для создания и редактирования отчётов — дизайнер отчётов Fast Reports. С его помощью вы можете формировать профессиональные отчёты по вашим данным: списки клиентов, отчёты по продажам, аналитические сводки и многое другое.

✅ Возможности дизайнера отчётов:
• Визуальное создание шаблонов отчётов без программирования
• Подключение к данным CRM (клиенты, сделки, звонки)
• Экспорт отчётов в популярные форматы: PDF, Excel, Word, HTML, изображения
• Предварительный просмотр перед печатью
• Сохранение и загрузка шаблонов
📌 Что такое Fast Reports?
Fast Reports — это профессиональная библиотека для генерации отчётов, используемая в тысячах приложений по всему миру. В состав My-Business CRM встроен полнофункциональный визуальный дизайнер, который позволяет создавать отчёты любой сложности без навыков программирования [citation:10].

🚀 Запуск дизайнера отчётов

1 Откройте раздел "Отчёты" в CRM
  • Запустите My-Business CRM
  • В главном меню найдите раздел "Отчёты" или "Дизайнер отчётов"
  • Нажмите на кнопку "Создать новый отчёт" или "Редактировать" для существующего шаблона
2 Откройте окно дизайнера
  • После нажатия откроется окно дизайнера отчётов Fast Reports
  • Интерфейс дизайнера включает:
    — Панель инструментов — для работы с отчётом (создание, открытие, сохранение, предпросмотр)
    — Область дизайна — рабочая область для размещения элементов отчёта
    — Окно данных — доступные поля из базы данных CRM
    — Инспектор свойств — настройка свойств выбранных элементов

📝 Создание нового отчёта

1 Выберите источник данных
  • В окне дизайнера перейдите на вкладку "Data" (Данные)
  • В дереве данных выберите нужные поля CRM:
    — Клиенты (ФИО, телефон, дата создания, воронка, стадия и т.д.)
    — Сделки
    — Звонки
  • Перетащите нужные поля мышью на рабочую область отчёта
💡 Совет: Для создания таблицы с данными используйте компонент "Table" или "Data Band" — он автоматически повторяется для каждой записи из базы данных.
2 Добавьте элементы оформления
  • Используйте панель инструментов слева для добавления элементов:
    — Text — текстовые поля, заголовки, подписи
    — Picture — изображения (логотип компании)
    — Line, Shape — линии и фигуры для оформления
    — Table — таблицы для структурированных данных
    — Subreport — вложенные отчёты
  • Настройте внешний вид через инспектор свойств (клавиша F11)
3 Настройте структуру отчёта
  • Используйте полосы (Bands) для организации структуры:
    — Report Title — заголовок отчёта (печатается один раз в начале)
    — Page Header — верхний колонтитул страницы
    — Data Header — заголовок таблицы данных
    — Data — основная полоса с данными (повторяется для каждой записи)
    — Data Footer — итоги по группе данных
    — Page Footer — нижний колонтитул страницы
    — Report Summary — общие итоги в конце отчёта
4 Предварительный просмотр
  • Нажмите кнопку Preview (Предпросмотр) на панели инструментов или клавишу Ctrl+P
  • В режиме предпросмотра вы можете увидеть, как будет выглядеть готовый отчёт
  • При необходимости вернитесь к редактированию и внесите правки
5 Сохраните шаблон отчёта
  • Нажмите Save или Save as на панели инструментов (или Ctrl+S)
  • Укажите имя файла шаблона (расширение .frx или .fr3)
  • Шаблоны хранятся в папке с программой и доступны для дальнейшего использования

📤 Экспорт отчётов

Дизайнер Fast Reports поддерживает экспорт в следующие форматы:
PDF Microsoft Excel (XLSX) Microsoft Word (DOCX) HTML CSV XML JSON PNG JPEG TIFF RTF TXT
Как экспортировать отчёт
  • В режиме предпросмотра (Preview) нажмите кнопку "Экспорт" на панели инструментов
  • Выберите нужный формат экспорта (например, PDF, Excel)
  • При необходимости настройте параметры экспорта (качество, страницы, ориентация)
  • Нажмите "Сохранить" и выберите папку для сохранения файла
Печать отчёта
  • В режиме предпросмотра нажмите кнопку Print (Печать) или Ctrl+P
  • Выберите принтер и настройки печати
  • Нажмите "Печать"

🔧 Расширенные возможности

Группировка данных

  • Добавьте полосу Group Header (Заголовок группы)
  • В инспекторе свойств укажите поле для группировки (например, "воронка" или "стадия")
  • Добавьте полосу Group Footer (Нижний колонтитул группы) для итогов по группе
  • Пример: отчёт по клиентам, сгруппированный по воронке продаж

Подсчёт итогов

  • Добавьте в отчёт текстовое поле для итога
  • В инспекторе свойств задайте функцию:
    — [SUM(Поле)] — сумма значений
    — [COUNT(Поле)] — количество записей
    — [AVG(Поле)] — среднее значение
    — [MIN(Поле)] — минимум
    — [MAX(Поле)] — максимум
  • Разместите итоговое поле в полосе Report Summary или Group Footer
Пример: [COUNT(Клиенты.ФИО)] - общее количество клиентов

Фильтрация данных

  • Выделите полосу с данными (Data Band)
  • В инспекторе свойств найдите свойство Filter
  • Задайте условие фильтрации (например, [Клиенты.Стадия] = "Сделка")
  • Отчёт будет показывать только записи, соответствующие условию

Использование параметров

  • Перейдите в меню Report → Parameters
  • Создайте новый параметр (например, "ДатаНачала", "Стадия")
  • В отчёте используйте параметр: [Параметр1]
  • При формировании отчёта CRM попросит ввести значения параметров
  • Это позволяет создавать интерактивные отчёты

⚡ Быстрые шаблоны отчётов

В состав CRM входят готовые шаблоны отчётов:
  • Список клиентов — таблица со всеми клиентами
  • Отчёт по воронке продаж — группировка по стадиям
  • Аналитика по звонкам — статистика звонков за период
  • Карточка клиента — детальный отчёт по одному клиенту

Вы можете использовать эти шаблоны как основу и редактировать их под свои нужды.

❓ Частые вопросы

Нужны ли навыки программирования для создания отчётов?

Нет. Дизайнер отчётов Fast Reports имеет визуальный интерфейс — вы создаёте отчёты перетаскиванием элементов мышью, без написания кода [citation:2][citation:10].

Как добавить логотип компании в отчёт?
  • Добавьте компонент Picture на полосу отчёта
  • В инспекторе свойств выберите файл изображения (логотип)
  • Настройте размер и положение
Как сделать отчёт с итогами по месяцам?
  • Используйте группировку по полю "Дата" с настройкой формата по месяцу
  • Добавьте в группу итоговую строку с суммой или количеством
Где хранятся созданные мной шаблоны отчётов?
  • Шаблоны сохраняются в папку с программой My-Business CRM
  • Файлы имеют расширение .frx или .fr3
  • Вы можете скопировать их для переноса на другой компьютер
Можно ли экспортировать отчёт в Excel с сохранением форматирования?
  • Да, при экспорте в формат XLSX сохраняется структура таблиц, шрифты и заливки
  • В настройках экспорта можно выбрать "Excel 2007/2010" для наилучшей совместимости
Вызов справки
  • В окне дизайнера отчётов нажмите F1 для вызова встроенной справки Fast Reports
  • Там вы найдёте подробное описание всех компонентов и функций [citation:7]
