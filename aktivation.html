🔑 Активация лицензионного ключа

После приобретения лицензии вы получили лицензионный ключ — который позволит использовать программу без окна, с предложением активировать программу. Активация происходит без подключения к интернету.

✅ Что вас ждёт после активации?
• Приоритетная техническая поддержка
• Персональные консультации
• Отсутствие информационных сообщений о покупке лицензии
📌 Перед началом:
Рекомендуем приобрести ключ, чтобы деактивировать окно с предложением купить лицензию. Он выглядит так:
XXXXX-XXXXX-XXXXX-XXXXX-XXXXX
После приобретения лицензии, ключ отобразится на экране, так-же есть возможность скачать TXT, с ключом.

📝 Пошаговая инструкция активации

1 Запустите My-Business CRM
  • Откройте программу через ярлык на рабочем столе или в меню "Пуск"
  • При запуске программы появится окно с предложением ввести лицензионный ключ
2 Введите лицензионный ключ
  • В открывшемся окне найдите поле для ввода ключа
  • Аккуратно введите полученный ключ (можно скопировать и вставить)
  • Важно: Ключ чувствителен к регистру, вводите его точно как в письме
  • Дефисы в ключе вводятся обязательно
3 Нажмите кнопку "Активировать"
  • После ввода ключа нажмите кнопку "Активировать" или "OK"
  • Активация происходит мгновенно, интернет не требуется
4 Проверьте статус лицензии
  • После активации информационные сообщения о покупке лицензии больше не будут появляться
  • При запуске программы окно с предложением ввести ключ больше не показывается

⚠️ Возможные ошибки и их решение

Ошибка: "Неверный лицензионный ключ"
  • Проверьте правильность ввода ключа — возможно, ошибка в символе
  • Убедитесь, что ключ введён без лишних пробелов в начале или конце
  • Попробуйте скопировать ключ из письма и вставить (Ctrl+V)
  • Проверьте, что дефисы введены правильно

🔍 Как проверить статус лицензии?

  • При запуске программы отсутствует окно с предложением ввести ключ
  • Отсутствуют информационные сообщения о покупке лицензии

🔄 Перенос лицензии на другой компьютер

Как перенести лицензию?
  • Лицензионный ключ можно использовать неоднократно, в пределах одной организации

❓ Частые вопросы

Нужен ли интернет для активации?

Нет. Активация происходит полностью в автономном режиме. Интернет-соединение не требуется.

Нужно ли активировать лицензию после обновления программы?

Нет. При обновлении программы лицензия сохраняется. Вам не нужно вводить ключ заново.

Сколько раз можно активировать ключ?
  • Ключ можно использовать неограниченное количество раз, в пределах одной организации
  • Не передавайте ключ третьим лицам
Где появляется окно для ввода ключа?
  • При запуске программы — если лицензия не активна
  • При инициализации звонка — при попытке совершить звонок
  • Также в меню Настройки → Лицензия или О программе
Будет ли работать лицензия на Linux через Wine?
  • Да, программа работает через Wine
  • Активация проходит аналогично Windows
  • Техническая поддержка для Linux ограничена
🛡️ Важно: Лицензия предназначена для использования одним пользователем (ИП или физическим лицом). Передача ключа третьим лицам запрещена лицензионным соглашением.
