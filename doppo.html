📦 Рекомендуемый пакет программ для My-Business CRM

Для полноценной работы с My-Business CRM рекомендуем установить дополнительные программы. Они расширяют функционал CRM и делают работу более комфортной.

✅ Все программы в этом списке — бесплатные и проверенные.
Они дополняют CRM, но не являются обязательными для её работы. Выбирайте то, что нужно именно вам.
📞

Телефония (SIP-клиенты)

Для совершения звонков из CRM необходим SIP-клиент. Рекомендуем следующие программы:

🥭
Mango Talker
Бесплатный SIP-клиент от Mango Office. Простая настройка — достаточно авторизации в учётной записи. Рекомендуемое решение.
Скачать → Инструкция →
📞
MicroSIP
Лёгкий бесплатный SIP-клиент. Не требует установки, работает в портативном режиме. Требует ручной настройки SIP.
Скачать → Инструкция →
📱
Zoiper
Популярный SIP-клиент с удобным интерфейсом. Есть бесплатная версия для Windows.
Скачать →
💚
Linphone
Бесплатный SIP-клиент с открытым исходным кодом. Поддержка Windows, Linux, macOS.
Скачать →
💬

Мессенджеры

Для отправки сообщений клиентам из CRM установите следующие программы:

✈️
Telegram Desktop
Официальное приложение Telegram для Windows. Требуется установочная версия (портативная не поддерживается).
Скачать → Инструкция →
💚
WhatsApp
WhatsApp для телефона + WhatsApp Web в браузере. Для работы с CRM нужна веб-версия.
Скачать для телефона → WhatsApp Web →
📊

Офисные приложения

Для работы с отчётами и выгруженными данными:

📗
Microsoft Excel
Профессиональный редактор электронных таблиц. При выгрузке данных из CRM файл открывается автоматически.
Подробнее →
📊
LibreOffice Calc
Бесплатная альтернатива Excel. Открывает CSV и XLSX файлы.
Скачать →
📝
ONLYOFFICE
Бесплатный офисный пакет для работы с документами и таблицами.
Скачать →
🗄️

Базы данных (для MySQL версии)

Для работы с MySQL версией CRM:

🐬
MySQL Server
Сервер баз данных для командной работы. Требуется для MySQL версии CRM.
Скачать → Инструкция →
🖥️
OpenServer / XAMPP
Готовые сборки для локального сервера. Включают MySQL, phpMyAdmin.
OpenServer → XAMPP →
🔧
phpMyAdmin
Веб-интерфейс для управления MySQL. Позволяет делать резервные копии.
Скачать →
🛠️

Adobe Acrobat Reader
