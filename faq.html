❓ Частые вопросы (FAQ)

Нужно ли покупать лицензию для работы программы? ▼
Нет, лицензия приобретается добровольно. Программа работает без функциональных, временных или количественных ограничений даже без лицензии. Лицензия — это способ поддержать разработчика и получить дополнительные преимущества (приоритетная поддержка, эксклюзивные функции, отключение сообщений).
Какую версию базы данных выбрать — SQLite или MySQL? ▼
  • SQLite — для одного пользователя, не требует настройки сервера, данные хранятся в файле.
  • MySQL — для командной работы (2+ пользователей), требует установки MySQL Server.
Если вы работаете один — рекомендуем SQLite. Если планируете работать с коллегами — выбирайте MySQL.
Как совершать звонки из CRM? ▼
  1. Установите Mango Talker или MicroSIP
  2. Войдите в учётную запись Mango Office (или настройте SIP в MicroSIP)
  3. В карточке клиента нажмите на иконку телефона рядом с номером
  4. Звонок начнётся автоматически!
Телефония доступна бесплатно.
Как отправить сообщение клиенту в Telegram или WhatsApp? ▼
  • Telegram: установите Telegram Desktop и авторизуйтесь. В карточке клиента нажмите на иконку Telegram.
  • WhatsApp: при первом нажатии на иконку откроется WhatsApp Web — отсканируйте QR-код телефоном. После этого чат будет открываться автоматически.
Номер телефона должен быть указан в международном формате (+7...).
Могу ли я работать в команде, не покупая лицензию? ▼
Да, вы можете использовать MySQL версию для совместной работы без лицензии. Единственное условие — каждый сотрудник должен иметь установленную программу и доступ к общей базе данных MySQL. Лицензия влияет только на дополнительные преимущества, но не на функциональность.
Где скачать программу? ▼
Нажмите кнопку «Скачать» в верхней части главной страницы. Доступны две версии:
  • SQLite версия — для одного пользователя
  • MySQL версия — для командной работы
Что делать, если возникла проблема? ▼
  1. Проверьте настройки программы и мессенджеров
  2. Посмотрите раздел «Возможные проблемы» в соответствующих инструкциях
  3. Напишите в техническую поддержку: company@mybusiness-crm.ru
Владельцы лицензии получают приоритетную поддержку.
