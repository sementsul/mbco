Интеграция мессенджеров

Общайтесь с клиентами в Telegram и WhatsApp прямо из карточки клиента

✅ Функция доступна бесплатно! Отправляйте сообщения клиентам в мессенджерах одним нажатием кнопки. Никакой сложной настройки не требуется.
📌 Как это работает?
В карточке каждого клиента есть кнопки Telegram и WhatsApp. При нажатии на кнопку автоматически открывается чат с клиентом в соответствующем мессенджере — по номеру телефона, указанному в карточке. Вам нужно только установить и авторизоваться в мессенджерах.
📱
Telegram

📥 Что нужно для работы?

  • Установить Telegram Desktop (официальное приложение для компьютера)
  • Авторизоваться в приложении (войти в свой аккаунт)

🚀 Как настроить?

  • Скачайте и установите Telegram Desktop с официального сайта
  • Запустите приложение и войдите в свой аккаунт (по номеру телефона)
  • Всё! My-Business CRM автоматически обнаружит Telegram
⬇️ Скачать Telegram Desktop
💡 Важно: Для отправки сообщения клиенту его номер телефона должен быть привязан к аккаунту Telegram (то есть у клиента должен быть зарегистрирован Telegram по этому номеру).
💚
WhatsApp

📥 Что нужно для работы?

  • Браузер (Chrome, Firefox, Edge или любой другой)
  • Аккаунт WhatsApp на телефоне

🚀 Как настроить?

  • Убедитесь, что WhatsApp установлен и авторизован на вашем телефоне
  • При первом нажатии на кнопку WhatsApp в карточке клиента откроется WhatsApp Web
  • Отсканируйте QR-код камерой телефона (как при обычном входе в WhatsApp Web)
  • После авторизации чаты будут открываться автоматически
📱 Примечание: WhatsApp Web работает через браузер и требует активного подключения телефона к интернету. Достаточно авторизоваться один раз — браузер запомнит сессию.

📞 Как это выглядит в CRM?

В карточке клиента имеются иконки мессенджеров
🖼️ [Карточка клиента]
  • Telegram — открывается Telegram Desktop с готовым чатом с клиентом
  • WhatsApp — открывается WhatsApp Web в браузере с чатом с клиентом
  • Номер телефона автоматически подставляется из карточки клиента

⚠️ Возможные проблемы и решения

  • Telegram не открывается: Убедитесь, что Telegram Desktop установлен и запущен
  • Чат не создаётся: Проверьте, что у клиента есть аккаунт в этом мессенджере
  • WhatsApp Web не открывается: Возможно, сессия истекла — отсканируйте QR-код заново
  • Номер не подставляется: Проверьте формат номера в карточке клиента (международный формат +7...)
💡 Совет: Для корректной работы указывайте номера телефонов в карточке клиента в международном формате (например, +79991234567). Это гарантирует правильное распознавание номера в мессенджерах.
