Версия с поддержкой MySQL

Версия для совместной работы в команде. Использует базу данных MySQL — идеально для организации, где несколько сотрудников работают с одной CRM.

✅ Бесплатно! Программа не имеет функциональных, временных или количественных ограничений. Лицензия приобретается только по вашему желанию — в знак поддержки разработчика.

📌 Что такое версия с MySQL?
MySQL — это серверная база данных, которая позволяет нескольким пользователям одновременно работать с CRM через локальную сеть или интернет. Отлично подходит для команд от 2 до 50 человек.

⚙️ Требования для версии с MySQL:

  • Установленный MySQL Server (версия 5.7 или выше)
  • Созданная база данных для CRM
  • Права на подключение к БД (логин/пароль)
  • При необходимости — настройка сетевого доступа

💡 Совет: Если вы работаете один и не планируете командную работу, рекомендуем версию с SQLite — она не требует настройки сервера.

🔧 Перед установкой:
Убедитесь, что MySQL Server установлен и запущен. Во время установки программы вам потребуется ввести данные для подключения к базе данных (хост, порт, имя БД, логин, пароль).
💡 О лицензии:
  • Программа работает без ограничений даже без лицензии
  • Лицензия приобретается добровольно, если программа вам понравилась
  • Лицензия даёт: приоритетную поддержку, эксклюзивные функции, отключение информационных сообщений
