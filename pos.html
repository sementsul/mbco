💻 Поддерживаемые операционные системы

My-Business CRM разработана для операционной системы Microsoft Windows 64-bit (x64). Это единственная платформа, на которой гарантируется полноценная и стабильная работа программы.

❗ Важно! Программа не поддерживает 32-битные (x86) версии Windows. Установка возможна только на 64-битные системы.
✅ Официально поддерживается:
Microsoft Windows 7 (x64), Windows 8 (x64), Windows 10 (x64), Windows 11 (x64). Версии Windows Server не тестировались, но работоспособность возможна.

🪟 Поддерживаемые версии Windows (только 64-bit)

🪟
Windows 7
Поддерживается (x64)
Только 64-битная версия с установленным Service Pack 1. 32-битная версия не поддерживается.
🪟
Windows 8 / 8.1
Поддерживается (x64)
Только 64-битная версия. 32-битная не поддерживается.
🪟
Windows 10
Поддерживается (x64)
Рекомендуемая версия. Требуется 64-битная система (Home, Pro, Enterprise).
🪟
Windows 11
Поддерживается (x64)
Полностью совместима, только 64-битная версия.
⚠️ 32-битные системы (x86) НЕ ПОДДЕРЖИВАЮТСЯ!
Если у вас установлена 32-битная версия Windows, вы не сможете установить My-Business CRM. Для работы программы требуется 64-битная архитектура.

⚠️ Другие операционные системы

❗ Важно! My-Business CRM не поддерживает macOS, Linux, Android, iOS и другие операционные системы, отличные от Windows 64-bit.
🍎
macOS
Не поддерживается
Программа не работает на macOS. Запуск через Parallels или Boot Camp не гарантируется.
🐧
Linux
Не поддерживается
Нативная версия для Linux отсутствует. Работа через WINE не гарантирует полноценную работу.
📱
Android / iOS
Не поддерживается
Мобильная версия не планируется. Программа разработана только для настольных ПК.

🍷 О работе через WINE (Linux / macOS)

⚠️ WINE не гарантирует полноценную работу программы!
Теоретически возможен запуск My-Business CRM через WINE на Linux или macOS, однако:
  • Не все функции могут работать корректно
  • Возможны проблемы с отображением интерфейса
  • Интеграция с SIP-клиентами может не работать
  • Мессенджеры (Telegram, WhatsApp) могут открываться некорректно
  • Техническая поддержка не оказывается для пользователей WINE
Рекомендация: Используйте программу только на официально поддерживаемой платформе — Windows 64-bit.

⚙️ Системные требования

  • Операционная система: Windows 7/8/10/11 (только x64, 64-bit)
  • Процессор: 1 ГГц или быстрее (x64)
  • Оперативная память: от 2 ГБ (рекомендуется 4 ГБ)
  • Свободное место на диске: от 200 МБ для установки + место для базы данных
  • Разрядность: только 64-bit (x64)
💡 Как узнать разрядность вашей Windows?
Панель управления → Система → Тип системы (64-разрядная или 32-разрядная). Если написано «32-разрядная» — установка программы невозможна.
