📦 Версии My-Business CRM

My-Business CRM доступна в двух редакциях — SQLite и MySQL. Обе версии имеют одинаковый функционал интерфейса, но различаются способом хранения данных и возможностями совместной работы.

✅ Важно: Обе версии полностью бесплатны для использования. Лицензия приобретается добровольно и даёт дополнительные преимущества, не ограничивая функционал программы.

📊 Сравнение версий

📁
SQLite
Для одного пользователя
  • Данные хранятся в одном файле (.db)
  • Не требует установки сервера
  • Идеально для одного сотрудника
  • Можно использовать на флешке
  • Поддержка сетевой папки
⬇️ Скачать SQLite
🐬
MySQL
Для командной работы
  • Данные хранятся на сервере MySQL
  • Требуется установка MySQL сервера
  • Совместная работа нескольких сотрудников
  • Высокая производительность при большом объёме данных
  • Работа через локальную сеть или интернет
⬇️ Скачать MySQL

🤔 Какую версию выбрать?

Выберите SQLite версию, если:
  • Вы работаете один (ИП, самозанятый, фрилансер)
  • Не хотите настраивать сервер баз данных
  • Хотите хранить данные на флешке или в облачном хранилище
  • Планируете использовать CRM на одном компьютере
Выберите MySQL версию, если:
  • У вас работает команда из 2+ сотрудников
  • У вас уже есть MySQL сервер (или вы готовы его установить)
  • Нужен централизованный доступ к данным из разных мест
  • Планируете обрабатывать очень большие объёмы данных
💡 Совет: Если вы не уверены, начните с SQLite версии. Она не требует настройки, и вы всегда сможете перейти на MySQL версию позже (потребуется перенос данных).

🔄 Совместимость и обновления

  • Windows: 7, 8, 10, 11 (x86 и x64)
  • Linux: через Wine (ограниченная поддержка)
  • MySQL версия: требует MySQL 5.7 или выше / MariaDB 10.2+
  • SQLite версия: не требует дополнительного ПО
⚠️ Важно о совместимости:
  • Версия с SQLite не может напрямую подключиться к MySQL базе
  • Версия с MySQL не может работать с файлом SQLite
  • Для перехода с SQLite на MySQL необходим экспорт/импорт данных через CSV
