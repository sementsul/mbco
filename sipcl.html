📞 Поддерживаемые SIP-клиенты

My-Business CRM поддерживает любые SIP-клиенты, которые могут выполнять инициализацию звонка по протоколу tel://. Это означает, что вы можете использовать практически любой SIP-софтфон для совершения звонков прямо из карточки клиента.

✅ Как это работает?
При нажатии на иконку телефона в карточке клиента CRM формирует ссылку с номером телефона по протоколу tel:// и передаёт её в операционную систему. Windows, в свою очередь, открывает эту ссылку в программе, которая зарегистрирована как обработчик этого протокола. Если ваш SIP-клиент поддерживает такой вызов и зарегистрирован в системе — звонок начнётся автоматически.
📌 Что такое протокол tel://?
Протокол tel:// — это стандартный URI-схема для телефонных номеров. При переходе по ссылке вида tel://+79991234567 система открывает приложение, которое зарегистрировано для обработки таких ссылок (обычно это Skype, MicroSIP, Mango Talker или другой SIP-клиент).
Пример ссылки, которую формирует CRM:
tel://+79991234567

✅ Какие SIP-клиенты поддерживаются?

Поддерживаются все SIP-клиенты, которые умеют обрабатывать протокол tel:// или могут быть зарегистрированы в системе как обработчик этого протокола. Вот наиболее популярные и проверенные решения для Windows:

🔧 Как проверить, работает ли протокол tel://?

Способ 1. Через командную строку
  • Нажмите Win + R, введите cmd и нажмите Enter
  • Введите команду: start tel://+79001234567
  • Если открывается ваш SIP-клиент и начинает звонок — протокол работает
Способ 2. Через браузер
  • Откройте браузер
  • В адресной строке введите: tel://+79001234567
  • Браузер должен предложить открыть ссылку в программе для звонков

⚙️ Как зарегистрировать SIP-клиент для протокола tel://?

  • Mango Talker — автоматически регистрируется при установке
  • MicroSIP — автоматически регистрируется при запуске (портативная версия)
  • Zoiper, X-Lite, Linphone, 3CX — обычно регистрируются при установке
  • Вручную: Параметры Windows → Приложения → Приложения по умолчанию → Выбрать стандартные приложения для протоколов → найти протокол TEL → выбрать нужный SIP-клиент
⚠️ Примечание: Некоторые SIP-клиенты (особенно портативные версии) могут не регистрироваться в системе автоматически. В этом случае, возможно, потребуется выполнить настройку вручную через параметры Windows или установить клиент через официальный установщик.

❓ Частые вопросы

Нужно ли настраивать CRM для работы с SIP-клиентом?

Нет. CRM автоматически отправляет номер по протоколу tel://. Вам не нужно указывать путь к программе или настраивать что-либо в CRM.

Что делать, если звонок не происходит?
  • Проверьте, что SIP-клиент запущен и авторизован
  • Проверьте, что SIP-клиент зарегистрирован для протокола tel://
  • Проверьте, что в карточке клиента номер указан в международном формате: +79991234567
Работает ли звонок через браузерные SIP-клиенты?

Браузерные SIP-клиенты (веб-версии) обычно не перехватывают протокол tel://. Для звонков из CRM используйте десктопные приложения.

Какой SIP-клиент лучше всего подходит?

Мы рекомендуем Mango Talker — он прост в настройке (достаточно авторизоваться в учётной записи Mango Office) и автоматически регистрируется в системе. Альтернатива — MicroSIP, если у вас уже есть SIP-аккаунт от другого провайдера.
