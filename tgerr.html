✈️ Почему не открывается чат в Telegram?

✈️

При нажатии на кнопку Telegram в карточке клиента должен открываться чат с этим клиентом в Telegram Desktop. Если этого не происходит, ниже перечислены возможные причины и способы их решения.

✅ Краткий ответ: My-Business CRM использует протокол tg:// для открытия чата. Проблемы чаще всего возникают из-за портативной версии Telegram или если протокол tg:// не зарегистрирован в системе.
⚠️ CRM НЕ РАБОТАЕТ с портативной версией Telegram!

Портативные версии Telegram (Portable, Telegram.exe без установки) не регистрируют протокол tg:// в системе. Без этого My-Business CRM не может передать номер телефона в Telegram.

Решение: Удалите портативную версию и установите официальную установочную версию с сайта desktop.telegram.org.

🔍 Как работает интеграция?

My-Business CRM использует протокол tg:// для открытия чата с клиентом. Формат ссылки выглядит так:

tg://resolve?phone=+79991234567

Когда вы нажимаете на кнопку Telegram в карточке клиента, CRM формирует такую ссылку и передаёт её системе. Windows должна передать эту ссылку Telegram Desktop, который откроет чат с указанным номером.

Для корректной работы необходимо, чтобы Telegram Desktop был установлен официальным установщиком и протокол tg:// был зарегистрирован в системе.

🔍 Основные причины и решения

1️⃣ Используется портативная версия Telegram

Причина: Портативная версия Telegram (exe-файл без установки) не регистрирует протокол tg:// в системе. My-Business CRM не может передать номер в Telegram.

  • Решение: Удалите портативную версию Telegram
  • Скачайте и установите официальную установочную версию с desktop.telegram.org
  • После установки авторизуйтесь в аккаунте
    • ⬇️ Скачать официальную версию Telegram
    2️⃣ Протокол tg:// не зарегистрирован в системе

    Причина: Даже при установленной официальной версии протокол может быть "сломан" или перехвачен другой программой.

  • Проверьте, что Telegram назначен обработчиком протокола tg://
  • Откройте Параметры Windows → Приложения → Приложения по умолчанию
  • Найдите "Выбрать стандартные приложения для протоколов"
  • Найдите протокол tg и убедитесь, что выбран Telegram Desktop
  • Если нет — нажмите и выберите Telegram из списка
    • 3️⃣ Telegram Desktop не запущен

    Симптомы: При нажатии на кнопку Telegram ничего не происходит, или появляется ошибка.

  • Убедитесь, что Telegram Desktop запущен (иконка в трее или панели задач)
  • Если программа закрыта — запустите её вручную
  • Рекомендуется добавить Telegram в автозагрузку, чтобы он всегда был запущен
    • 4️⃣ Не выполнен вход в аккаунт Telegram

    Симптомы: Telegram Desktop открывается, но чат не создаётся, программа показывает окно входа.

  • Откройте Telegram Desktop
  • Введите номер телефона и код подтверждения для входа в аккаунт
  • После авторизации чаты будут открываться корректно
    • 5️⃣ У клиента нет аккаунта в Telegram

    Симптомы: При нажатии на кнопку Telegram открывается пустой чат, или появляется ошибка, что пользователь не найден.

  • Убедитесь, что номер телефона клиента привязан к аккаунту Telegram
  • Вы можете проверить это — попробуйте найти клиента в Telegram вручную
  • Если аккаунта нет — предложите клиенту зарегистрироваться в Telegram
    • 6️⃣ Неправильный формат номера телефона

    Симптомы: Нажатие на кнопку Telegram не даёт результата, или открывается не тот чат.

  • Номер телефона в карточке клиента должен быть в формате 899999999999, без скобок и дифисов
  • Правильный пример: 899999999999
  • Неправильные примеры: 8(999)999-99-99, 8(999)1234567, (999)1234567
  • Исправьте формат номера через редактирование карточки
    • Правильный формат: 89991234567
    Неправильный формат: 8(999)123-45-67
    7️⃣ Блокировка антивирусом или файрволом

    Симптомы: При попытке открыть чат появляется ошибка или окно не открывается.

  • Проверьте настройки антивируса — возможно, он блокирует открытие ссылок tg://
  • Добавьте Telegram Desktop и My-Business CRM в исключения антивируса
  • Проверьте настройки брандмауэра Windows
    • 8️⃣ Telegram установлен из Microsoft Store

    Симптомы: При нажатии на кнопку Telegram ничего не происходит или открывается браузер.

  • Версия Telegram из Microsoft Store имеет ограничения для работы с внешними протоколами
  • Рекомендуется использовать версию с официального сайта desktop.telegram.org
  • Удалите версию из Store и установите официальную

    • 🛠️ Как восстановить протокол tg://?

    Способ 1: Переустановка Telegram
    • Полностью удалите Telegram через "Установка и удаление программ"
    • Удалите папку с остатками (если есть) — обычно %AppData%\Telegram Desktop
    • Скачайте свежую версию с официального сайта
    • Установите Telegram и авторизуйтесь
    • После этого протокол tg:// должен зарегистрироваться автоматически
    Способ 2: Назначение вручную через настройки Windows
    • Откройте Параметры Windows (Win + I)
    • Перейдите в Приложения → Приложения по умолчанию
    • Прокрутите вниз до пункта "Выбрать стандартные приложения для протоколов"
    • Найдите протокол tg
    • Нажмите на текущее приложение и выберите Telegram Desktop из списка
    Способ 3: Проверка через командную строку

    Выполните команду в командной строке (Win + R → cmd):

    assoc .tg

    Если протокол зарегистрирован, команда покажет ассоциацию. Также можно проверить открытие ссылки вручную:

    start tg://resolve?phone=+79001234567

    Если после этой команды открывается Telegram — протокол работает корректно. Если нет — проблема в системе.

    🛠️ Пошаговая диагностика

    1 Проверьте, какая версия Telegram установлена
    • Найдите Telegram в меню "Пуск"
    • Если Telegram просто как файл .exe в папке — это портативная версия
    • Если есть запись в "Установка и удаление программ" — установленная версия
    • Для работы нужна только установленная версия!
    2 Запустите Telegram и войдите в аккаунт
    • Запустите Telegram Desktop
    • Убедитесь, что вы авторизованы (виден список чатов)
    3 Проверьте формат номера клиента в CRM
    • Откройте карточку клиента в My-Business CRM
    • Проверьте поле "Телефон" — он должен быть в формате +79991234567
    4 Проверьте протокол tg://
    • Нажмите Win + R, введите cmd и нажмите Enter
    • Введите команду: start tg://resolve?phone=+79001234567
    • Если открывается Telegram — протокол работает
    5 Проверьте в CRM
    • Вернитесь в CRM и нажмите на кнопку Telegram
    • Должен открыться чат с клиентом
    ✅ Если чат открылся — всё работает!

    📌 Краткое резюме

    • Портативная версия Telegram НЕ работает — используйте только установочную
    • Протокол tg:// должен быть зарегистрирован — проверьте через командную строку
    • Telegram должен быть запущен — добавьте в автозагрузку
    • Номер телефона в карточке — в международном формате (+7...)
    • У клиента должен быть аккаунт в Telegram по этому номеру
    🚑 Если ничего не помогло:
    Обратитесь в техническую поддержку и укажите:
    • Версию Telegram Desktop (Справка → О программе)
    • Установлена ли версия из Store или портативная
    • Результат команды: start tg://resolve?phone=+79001234567
    • Формат номера телефона в карточке клиента
