📊 Выгрузка данных в Microsoft Excel

My-Business CRM позволяет быстро выгрузить информацию о контактах в Microsoft Excel (формат XLSX или CSV). Эта функция удобна для создания отчётов, анализа данных, передачи информации коллегам или подготовки рассылок.

✅ Преимущества выгрузки в Excel:
• Анализ данных с помощью сводных таблиц и диаграмм
• Печать списка клиентов
• Обмен данными с другими программами
• Создание резервных копий в удобном формате
• Фильтрация данных перед выгрузкой

📝 Пошаговая инструкция выгрузки

1 Откройте список контактов
  • Запустите My-Business CRM
  • Убедитесь, что вы находитесь в основном окне со списком контактов
  • При необходимости выберите нужную базу данных
2 Настройте фильтры (по желанию)
  • Используйте фильтры в верхней части списка для отбора нужных контактов
  • Вы можете отфильтровать по ФИО, телефону, воронке, стадии, дате добавления
  • Будут выгружены только те контакты, которые отображаются в списке после применения фильтров
🔍 Примеры фильтров:
Воронка: Продажи Стадия: Переговоры Дата: За последний месяц ФИО содержит: ООО
💡 Совет: Если не устанавливать фильтры, будет выгружен полный список всех контактов в текущей базе данных.
3 Нажмите кнопку "Выгрузить"
  • На верхней панели инструментов найдите кнопку "Выгрузить"
  • Нажмите на неё один раз
  • Программа автоматически сформирует файл с данными
4 Автоматическое открытие в Excel
  • После нажатия кнопки файл автоматически откроется в Microsoft Excel
  • Вам не нужно искать файл в папках — он откроется сам
  • Файл содержит все поля контактов: ФИО, телефон, email, воронку, стадию, дату создания, комментарии и т.д.
✅ Готово! Вы можете работать с данными в Excel: сортировать, фильтровать, строить диаграммы, сохранять файл на диск.

📋 Какие данные выгружаются?

  • ФИО — полное имя контакта
  • Телефон — номер телефона
  • Воронка — название воронки продаж
  • Стадия — текущая стадия сделки
  • Дата создания — когда контакт был добавлен в CRM
  • Дата последнего звонка — последнее взаимодействие

📌 Примеры использования выгрузки

Пример 1. Формирование отчёта по активным сделкам
  • Установите фильтр: Стадия = "Переговоры" или "Сделка"
  • Нажмите кнопку "Выгрузить"
  • В Excel откроются все активные сделки
  • Постройте сводную таблицу для анализа
Пример 2. Обзвон клиентов за последнюю неделю
  • Установите фильтр: Дата последнего звонка > 7 дней
  • Или: Не было звонков
  • Нажмите "Выгрузить" → список клиентов для обзвона готов
  • Распечатайте список или используйте прямо в Excel
Пример 3. Список клиентов для рассылки через мессенджеры
  • Установите фильтр: по воронке или стадии (например, "Тёплые клиенты")
  • Нажмите "Выгрузить"
  • В Excel скопируйте столбец "Телефон" и столбец "ФИО"
  • Используйте список для обзвона или рассылки через Telegram/WhatsApp
Пример 4. Анализ воронки продаж
  • Выгрузите полный список
  • В Excel постройте сводную таблицу по полю "Стадия"
  • Проанализируйте количество клиентов на каждом этапе
  • Оцените эффективность воронки

⚠️ Если Microsoft Excel не установлен

Что делать, если программа пишет ошибку или файл не открывается?
  • Убедитесь, что на компьютере установлен Microsoft Excel (2007 или новее)
  • Если Excel не установлен, вы можете использовать, OpenOffice
📁 Где найти выгруженный файл?
Файл автоматически сохраняется в следующую папку:
C:\Users\Ваше_имя\Downloads\

Или в папку с программой, в зависимости от настроек.

❓ Частые вопросы

Можно ли выгрузить только определённые поля?
  • В текущей версии выгружаются все поля контакта
  • После выгрузки в Excel вы можете удалить ненужные столбцы вручную
Какой формат файла используется?
  • По умолчанию выгрузка происходит в формате .xlsx (Microsoft Excel)
  • Этот формат поддерживает все современные версии Excel
Сколько контактов можно выгрузить за раз?
  • Ограничений нет — выгружается весь список, который отображается в CRM
  • При очень большом количестве контактов (100 000+) выгрузка может занять несколько секунд
Сохраняется ли файл на диск?
  • Да, файл сохраняется во временную папку или в папку "Загрузки"
  • После работы в Excel не забудьте сохранить файл, если хотите его оставить
Почему файл не открывается в Excel?
  • Возможные причины: Excel не установлен, используется старая версия Excel (2003 и ниже), повреждён файл
  • Решение: установите Excel 2007 или новее, либо используйте бесплатный аналог LibreOffice
Можно ли настроить автоматическую выгрузку по расписанию?
  • В текущей версии автоматическая выгрузка по расписанию не предусмотрена
  • Выгрузка выполняется вручную по нажатию кнопки

4️⃣ Файл автоматически откроется в Microsoft Excel
5️⃣ Работайте с данными: анализируйте, сохраняйте, печатайте
